Interface

Uma interface é um ponto de interação ou comunicação entre dois sistemas, dispositivos ou entidades, permitindo que eles atuem ou se comuniquem entre si.

Ux

UX (User Experience) é o que o usuário sente ao interagir com a ferramenta, se ele sente dificuldade de utilizar ela ou consegue usá-la de forma intuitiva.

ui

UI (User Interface) é saber se sua ferramenta será clara e intuitiva de se utilizar, se ela tem aspectos que melhoram sua interação e compreensão do usuário.

Front End

O front-end é o responsável pelo visual do site, sem ele os sites seriam apenas blocos cinzas.

É formado por arquivos html com tags html que definem a estrutura dos elementos, css que utilizam os elementos e javascript que realiza as interações dos elementos com o usuário.

html

O html é responsável por definir a estrutura dos elementos com tags html, no caso de um formulário temos a tag <form> para exibi lo. Tags como <input> trabalham juntos com atributos de type (tipo), ex. type=”text” pode ser usado para definir um campo de texto, ou ser mais específico utilizando o type="email" que configura o campo de dados a enviar para o banco de dados apenas sequência de caracteres correspondente a email válidos, ex exemplo@gmail.com.

O <button> com <type="submit"> envia os dados para o servidor.

Tente digitar um email errado

css

O arquivo css utiliza propriedades para estilizar elementos html, e para que a estilização seja aplicada é necessário referenciar no html o arquivo css.

<link rel="stylesheet" href="style.css">>

As propriedades css são responsáveis por definir por exemplo cores que em sua maioria são representadas com sequências hexadecimais, tamanho de elementos, fontes, maneiras de como os elementos irão se posicionar, responsividade e interações básicas com o usuário dado ao passar o mouse em cima ou clicar no botão e tendo como resultado a mudança de cor ou a execução de uma animação.

Descore Seu botão

background: red

}

js

Arquivos javascript são aquele responsáveis por criar interações entre elementos e ações do usuário no exemplo anterior os botões que alteram as propriedades css se utilizam do js para poder modificar as propriedades css de elementos pré determinados.

Juntos com o elemento X, a criatividade, temos o front end de uma página web.

sistema operacional

O Sistema Operacional (SO), é o responsável por criar uma interface gráfica que softwares/ aplicativos como navegadores possam acessar recursos do hardware do dispositivo.

Sem ele ao utilizar qualquer dispositivo só visualizar a tela apagada no monitor.

kernel

Para que possa manipular os componentes do dispositivo, é crucial o uso do Kernel que quando o dispositivo é ligado, é o responsável por identificar os componentes do hardware, e gerenciar os processos do processador e memória, para que sejam manipulados pela parte gráfica e de APIs que disponibilizam parte do controle dos componentes físicos.

componentes
